# Aries network with mobile and desktop edge agents


## run
1. add .env to ./desktop and ./mobile
2. add SEED (32 chars) to .env
3. add ALIAS to .env
4. add ROLE to .env


template .env:
```bash
SEED=12345678901234567890123456789012
ALIAS=TEST_AGENT
ROLE=TRUST_ANCHOR
API_KEY=34785687346537846537846 -> only for ./mobile/.env
```
